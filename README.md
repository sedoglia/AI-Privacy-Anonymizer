# AI-Privacy-Anonymizer
Uno strumento Python che consente di caricare file di vario formato contenenti dati personali, rilevare e mascherare automaticamente le PII (Personally Identifiable Information), e produrre in output un file anonimizzato con la stessa estensione dell'originale, pronto per essere caricato su chatbot AI senza rischi di data leakage.
